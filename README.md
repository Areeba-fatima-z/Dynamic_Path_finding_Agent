# Dynamic Pathfinding Agent

A grid-based pathfinding visualization tool implementing Greedy Best-First Search and A* algorithms with dynamic obstacle handling.

## Features
- Dynamic grid sizing
- Multiple heuristics (Manhattan, Euclidean, Diagonal)
- Real-time path visualization
- Dynamic obstacle spawning
- Interactive map editor
- Performance metrics display
